Course Objectives: An overall focus in this course will be based on a practical approach to design, implementation, and management of database management systems. The general objectives of the course include:  build theoretical foundation for database management systems, understand database management systems role in emerging applications, learn database modeling techniques, study different database models, study relational algebra, learn query writing using SQL, and database analysis and design.

 

Homework:  Reading assignment only, no points.

 

Project:  There is a mandatory project based on a real world database application selected by the student. Each student will work alone on the project to complete the tasks outlined below. As the project contains 45% of the grade, it is important that you spend enough time to achieve the project goals.  The project goals are:

-         Understand a real world problem, design and implement it in Oracle DBMS 

-    Make your project complete

-         Demonstrate your project successfully to the instructor. 

 

Project Description:  Choose a problem that consists of at least 6 entities in your model. Develop an E-R model or a UML model for this database. Perform a database design with constraints, attributes, and relationships. Implement this model in the Triton based system using Oracle DBMS.  Write reasonable number of complex queries that can be run in the SQL prompt mode. Run the same number of queries using a Java JDBC or a C++ ODBC program. If possible, run your database through your Web page (extra points 10%). 

 

Project Tasks and grades (total points for the project 45):

1.      Data model          10%

2.      Database design   10%

3.      Database implementation                          20%

4.      Running complex queries in SQL prompt 30%

5.      Running complex queries in Java JDBC, or C++ ODBC program 20%

6.      Documentation and Report 10%.

 

 

If there is no actual working of any piece in your project, your entire project grade will be zero. Grading for the above steps is strictly enforced.
