Course Objectives: An overall focus in this course will be based on a practical approach to design, implementation, and management of operating systems. The general objectives of the course include:  build theoretical foundation for operating systems, understand operating systems role in emerging applications, learn process, memory, and file management techniques, study different operating system models, understand at least one commercial operating system through a case study, and get some hands on experience in building OS components.

 

Homework:  Reading assignments only, no points.

 

Project:  There is a mandatory project based on the real world operating system components. Each student will work alone on the project to complete the tasks outlined below. As the project contains 45% of the grade, it is important that you spend enough time to achieve the project goals.  The project goals are:

-         Understand bare PC computing environment and its OS related structures

-         Make your project complete

-         Demonstrate the project successfully to the instructor. 

 

Project Tasks and grades (total points for the project 45):

1.      Understand batch files      5%

2.      Understand the bare PC compiling and linking environment  5%

3.      Successfully run the boot and data disks   5%

4.      Understand the I/O interfaces to hardware  5%

5.      Understand task interfaces, should be able to explain how it works   10%

6.      Pick a reasonably complex C++ program (which runs at least 10 minutes on the PC); run the above application on a Windows machine and measure the run time  10%

7.      Make the above program run on the bare PC and measure the run time  30%

8.      Compare the run times on Windows and on bare PC (some graphs) with details   5%

9.      Provide analysis and explanation for the differences in run times 5%

10.  Present your knowledge and demonstration of the program and results in the lab to the instructor 10%

11.  Documentation and report 10%.

 

 

If there is no actual working of any piece in your project, your entire project grade will be zero. Grading for the above steps is strictly enforced.
