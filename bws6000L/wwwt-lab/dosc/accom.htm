Accomplishments: Since this work started almost 5 years ago, we made significant milestones as listed below:

  • Learned how to work with a bare PC by by-passing the existing OS.

  • Developed an interface to PC that enables us to test the DOSC paradigm

  • Developed device drivers for Ethernet cards 3COM 509B, and 3COM905CX, Intel internal NIC 82540EM

  • Developed a device driver for AC-97 Sound Chip on the motherboard

  • Developed a comprehensive C++ API to build application objects (AOs)

  • Developed and tested successfully AOs for hello, bubble sort, TCP/UDP messages, Email Client, POP Client, Web Server (with multi-tasking), Web client, VoIP softphone, and secure VoIP softphone

  • Ported security protocols such as SHA-1, AES, and RSA to run on bare PC

  • Built a fully functional Web server running at http://dosc.towson.edu on a bare PC.  

  • Built Email Servers and Clients that run bare PCs

  • Built TLS on Web server

  • Built IP-Sec for bare PC applications

  • Produced 6 doctoral thesis related to bare machine computing
