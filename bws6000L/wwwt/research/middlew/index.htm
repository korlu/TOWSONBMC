Intro

Middleware or data integrators are hard to define terms. However, they are present in almost every computer system and applications. We study this problem with reference to its understanding, architecture, application domains, existing solutions, future direction and its relationship with similar problems in multidatabase and heterogeneous database applications. One should aim for a bullet proof solution which can be applied to many application domains!
